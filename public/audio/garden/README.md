# Little Garden generated voice assets

These interim offline clips were generated at build time on 2026-07-18 with `edge-tts` 7.2.8 and `en-US-AvaNeural`, using a `-10%` rate and `-2Hz` pitch.

| File stem     | Spoken line           |
| ------------- | --------------------- |
| `tap-cloud`   | “Tap the rain cloud.” |
| `tap-sun`     | “Tap the warm sun.”   |
| `flower-grew` | “The garden grew!”    |

Each cue is a trimmed, -18 LUFS/-2 dB true-peak, 48 kHz mono OGG Vorbis/MP3 pair. Runtime playback uses only bundled files and stable keys, so human recordings can replace these without code changes.
